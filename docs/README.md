# Documentation

This folder holds product-level and feature-level documentation for Life Manager.

## Contents

- [product-goals.md](product-goals.md) — the overall vision, principles, and roadmap.
- [features/](features/) — specifications for larger features. Add one file per
  feature before implementing it.

## Adding a Feature Spec

1. Copy [features/_template.md](features/_template.md) to a new file, e.g.
   `features/weather-summary.md`.
2. Fill in the goal, scope, and acceptance criteria.
3. Implement against the spec, updating it as decisions change.
